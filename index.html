Projects

  • purescript-difflists — a Purescript library for using difference lists, which have constant-time appending on the left- and right-hand sides, but transform to ordinary Cons lists in linear time.
  • purescript-simple-request — a Purescript wrapper around parts of node's http and https module, focusing on making requests. Uses purescript-options to set options in the request.
  • purescript-reddit-api — a Purescript library for interacting with Reddit's API via purescript-simple-request. Uses a monad transformer stack of StateT, EnvT and Aff. Automatically handles OAuth login, token renewal and rate limiting.
  • purescript-symbolic-differentiation — a Purescript library for symbolically computing derivatives of "real"-valued functions. Supports taking the Taylor expansion of a function to some finite number of terms.
  • purescript-markov-chains — a Purescript library for creating Markov chains from arbitrary orderable data structures, with extra features for strings.
  • purescript-mmorph — a port of Haskell's MMorph library for working with different monad transformer stacks.
  • SandScript — a lisp interpreter that runs on top of Node. This is the result of going through Write Yourself a Scheme, but adapting it for use in Purescript instead of Haskell. Differences in particular include Purescript's evaluation strategy, which matches Javascripts, making it necessary to change the way recursive S-expressions are parsed.
  • JARAW — an earlier library for interacting with Reddit's API, but more suitable for use with Javascript. Written with Iced Coffeescript.
  • RAADB — a library for using Reddit as a database, written in Javascript. Intended as a database replacement for hobbyist projects only.
